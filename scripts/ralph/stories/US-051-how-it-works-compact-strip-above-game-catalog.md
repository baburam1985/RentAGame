## Description

A compact 3-step process strip — 'Browse → Book Online → Delivered to You' — is displayed above the game grid and below the hero section. This orients first-time visitors before they engage with the catalog. It is a condensed, icon-driven version distinct from the existing full HowItWorks section at the bottom of the page.

## Acceptance Criteria

- [ ] A horizontal strip with exactly 3 steps renders above the game grid (after the hero, before the filters/grid)
- [ ] Each step shows an icon (Material Symbols), a short label (e.g. 'Browse'), and a one-line description
- [ ] The steps are: (1) Browse our games, (2) Book online in minutes, (3) Delivered to your door
- [ ] The strip is responsive: single row on desktop, stacked or scrollable on mobile (375px)
- [ ] The strip does not modify the existing HowItWorks component at the bottom of the page
